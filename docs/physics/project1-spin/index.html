    Magnetic resonance - Rabi's oscillation

    The system of Rabi oscillation #

    Neutral atom with spin 1/2 is immersed in the magnetic field of $\vec{B} = B_0 \hat{z}+B_1\left( \cos(\omega t)\hat{x}+\sin(wt)\hat{y}\right)$ where the magnetic field is rotating anti-clockwise with respect to z axis with angular frequency $\omega$. The the magnetic moment of the atom is,

    $$\begin{equation} \hat{\mu} = \frac{e}{m}\hat{S} = \frac{e}{m}\frac{\hbar}{2}\hat{\sigma} \end{equation}$$

    The Hamiltonian of the system is then,

    $$\begin{equation} \hat{H} = -\hat{\mu}\cdot\hat{B} = \frac{\hbar}{2} \begin{pmatrix} \Omega_0 & \Omega_1 e^{-iwt} \\ \Omega_1 e^{iwt} & -\Omega_0\end{pmatrix} \end{equation}$$

    where $\Omega_0 = \frac{eB_0}{m}$ and $\Omega_1 = \frac{eB_1}{m}$.

    Solving for Schrödinger equation #

    To see how a state evolve over time, we solve time-dependent Schrödinger equation,

    $$\begin{equation} i\hbar\frac{d}{dx}\ket{\psi}=\hat{H}\ket{\psi}=\frac{\hbar}{2}\begin{pmatrix} \Omega_0 & \Omega_1 e^{-iwt} \\ \Omega_1 e^{iwt} & -\Omega_0\end{pmatrix}\begin{pmatrix} a(t)\\b(t) \end{pmatrix} \end{equation}$$

    where $\ket{\psi} = a(t)\ket{\uparrow}+b(t)\ket{\downarrow}$ is an arbitrary state vector written in basis of eigen state of operator $\hat{S}^2$ and $\hat{S_z}$. Expanding it, we get a coupled equation,

    $$\begin{align} i\frac{d}{dt}a(t)=\frac{\Omega_0}{2}a(t)+\frac{\Omega_1}{2}e^{-iwt}b(t)\\ i\frac{d}{dt}b(t)=\frac{\Omega_1}{2}e^{iwt}a(t)+\frac{\Omega_0}{2}b(t) \end{align}$$

    Notice the time explicit dependence of the coefficient of each linear differential equations. This is because our Hamiltonian is not diagonalized and these off diagonal terms give time explicitly dependent terms(If we go back to the solid state physics, in the chapter tight binding model, off diagonal terms allow hamiltonian to evolve over time; electrons hop between atoms). We can get rid of the time explicit dependence by applying rotating operator to our state vector $\ket{\psi}$ such that,

    $$\begin{equation} \ket{\widetilde{\psi}} = \hat{R_z}\ket{\psi} = e^{-iwt\hat{S_z}/\hbar}\ket{\psi} \end{equation}$$

    state vector under rotation becomes,

    $$\begin{equation} \begin{pmatrix} \widetilde{a}(t)\\ \widetilde{b}(t) \end{pmatrix} = \begin{pmatrix} a(t)e^{iwt/2}\\ b(t)e^{-iwt/2} \end{pmatrix} \end{equation}$$

    $$\begin{align} a(t) = \widetilde{a}(t)e^{-iwt/2} \\ b(t) = \widetilde{b}(t)e^{iwt/2} \end{align}$$

    By subsituting equation 8 and 9 to our coupled equation, it becomes,

    $$\begin{align} i\frac{d}{dt}\widetilde{a} = -\frac{\Delta}{2}\widetilde{a}(t)+\frac{\Omega_1}{2}\widetilde{b}(t) \\ i\frac{d}{dt}\widetilde{b} = \frac{\Omega_1}{2}\widetilde{a}(t)+\frac{\Delta}{2}\widetilde{b}(t) \\ \end{align}$$

    where $\Delta = \omega - \Omega_0$.

    This coupled equation can be solved by either finding eigenvalue or elimination method.

    Solution is then,

    $$\begin{gather} \widetilde{a}(t) = C_+e^{i\Omega t/2}+C_-e^{-i\Omega t/2}\\ \widetilde{b}(t) = -\frac{\Omega-\Delta}{\Omega_1}C_+e^{i\Omega t/2}+\frac{\Omega+\Delta}{\Omega_1}C_-e^{-i\Omega t/2} \end{gather}$$ and $$\begin{gather} a(t) = \left (C_+e^{i\Omega t/2}+C_-e^{-i\Omega t/2}\right )e^{-iwt/2}\\ b(t) = \left (-\frac{\Omega-\Delta}{\Omega_1}C_+e^{i\Omega t/2}+\frac{\Omega+\Delta}{\Omega_1}C_-e^{-i\Omega t/2} \right )e^{iwt/2} \end{gather}$$

    Initial condition #

    The coefficient $C_+$ and $C_-$ are found by setting initial condition. The generalized value of the coefficient are,

    $$\begin{gather} C_+ = \frac{a(0)(\omega_1+\omega_2)-b(0)}{2\omega_1}\\ C_- = \frac{a(0)(\omega_1-\omega_2)+b(0)}{2\omega_1} \end{gather}$$

    where $\omega_1$ and $\omega_2$ are $\frac{\Omega}{\Omega_1}$ and $\frac{\Delta}{\Omega_1}$. If the state vector at t = 0 is spin up state, $C_+$ and $C_-$ are,

    $$\begin{gather} C_+ = \frac{\omega_1+\omega_2}{2\omega_1}\\ C_- = \frac{\omega_1-\omega_2}{2\omega_1} \end{gather}$$

    We need to take a note that we need to set the value of $a(0)$ and $b(0)$ such that the probability of $\braket{\psi(0)|\psi(0)}$ is equal to 1.

    Simulation # 

    For drawing '2D' plots
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
from numpy.linalg import *
from math import *

# Constants and parameters

w = 5 # Frequency for x(t) and y(t)
Omega0 = 5 #eB0
Omega1 = 1 #eB1
Delta = w-Omega0
Omega = np.sqrt(Delta**2 + Omega1**2)
w1 = Omega/Omega1
w2 = Delta/Omega1
t = np.linspace(0, 100, 10000)  # Time array range 100 is broken into 10000 pieces
# a and b
aa = ((w1+w2)/(2*w1))*np.exp(1j*Omega*t/2)+((w1-w2)/(2*w1))*np.exp(-1j*Omega*t/2)
bb = -(w1-w2)*((w1+w2)/(2*w1))*np.exp(1j*Omega*t/2)+(w1+w2)*((w1-w2)/(2*w1))*np.exp(-1j*Omega*t/2)
a = aa*np.exp(-1j*w*t/2)
b = bb*np.exp(1j*w*t/2)

#Create 2D plots
plt.subplot(311)
plt.grid()
plt.plot(t, a*np.conj(a)+b*np.conj(b), label = '<Ψ|Ψ>')
plt.xlabel('t')
plt.legend()

plt.subplot(312)
plt.grid()
plt.plot(t, a*np.conj(a), label = '|<↑|Ψ>|²')
plt.xlabel('t')
plt.legend()

plt.subplot(313)
plt.grid()
plt.plot(t, b*np.conj(b), label = '|<↓|Ψ>|²')
plt.xlabel('t')
plt.legend()

# Show the plot
plt.show()

    For drawing '3D' plots
import numpy as np
import matplotlib.pyplot as plt
from math import *

# Constants and parameters
w = 5 # Frequency for x(t) and y(t)
Omega0 = 5 #eB0
Omega1 = 1
Delta = w-Omega0
Omega = (Delta**2 + Omega1**2)**.5
w1 = Omega/Omega1
w2 = Delta/Omega1
T = 2*pi/Omega
t = np.linspace(0, 1*T, 100000)
# state vector
aa = ((w1+w2)/(2*w1))*np.exp(1j*Omega*t/2)+((w1-w2)/(2*w1))*np.exp(-1j*Omega*t/2)
bb = -(w1-w2)*((w1+w2)/(2*w1))*np.exp(1j*Omega*t/2)+(w1+w2)*((w1-w2)/(2*w1))*np.exp(-1j*Omega*t/2)

a = aa*np.exp(-1j*w*t/2)
b = bb*np.exp(1j*w*t/2)
#Coordinatizing expectation value of Spin operator
x = (np.conj(a)*b)+(np.conj(b)*a)
y = 1j*((-np.conj(a)*b)+(np.conj(b)*a))
z = (np.conj(a)*a)-(np.conj(b)*b)
# Create 3D plots
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot(x, y, z)

ax.axis('equal')
ax.set_xlim(-1, 1)
ax.set_ylim(-1, 1)
ax.set_zlim(-1, 1)

# Set labels for the axes
ax.set_xlabel('X(t)')
ax.set_ylabel('Y(t)')
ax.set_zlabel('Z(t)')
# Show the plot
plt.show()
